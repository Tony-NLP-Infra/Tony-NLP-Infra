# Bilingual NLP & Text Data Engineer JA/EN
**Bilingual NLP Analyst with Deep Linguistic Insights and Experience in Infrastructure.**

### Career Objective
Transitioning to **NLP Analysis** from **Infrastructure Engineering**. I have experience in building the servers that run the models, and the pipelines that clean the data, with a focus on **Voice of Customer (VoC)** and **Sentiment Trend Analysis**.

*   **Modeling:** Sentiment Analysis & Time-Series Forecasting using `RoBERTa`, `BERT`, and `Scikit-Learn`.
